## 设置
1. 安装node和git

2. 从github上复制代码
  ```
  git clone https://github.com/Tian508/discord-bot.git
  ```
3. .env.example改为.env,在.env中配置脚本所需参数
4. 安装运行脚本需要的依赖包,执行脚本
```
- npm install
- node monitor.js
```
